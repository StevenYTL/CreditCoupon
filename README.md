CreditCoupon

## 【APK】
CreditCoupon.apk

## 【開發工具】
* Android Studio─Java
* python - selenium 爬蟲套件，將網路上優惠資訊存進資料庫
* python - beautifulsoup4，解析html
* 利用CardView以及RecyclerView編排版面
* 利用Notification傳遞最新優惠資訊通知
* okhttp3抓取URL雲端資料、傳遞資料
* JSON檔案解析(org.json)
* BottomNavigationView底部功能表
* AsyncTask同步抓取雲端資料
